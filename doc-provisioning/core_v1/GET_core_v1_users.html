GET on core/v1/users

Description

https://api.groupcamp.com/core/v1/users

Return all the users of the account, filtered according to the GET arguments received.

GET parameters

Optional or required query parameters.

Name Type Description
state String Filter selected users on their state.
user_type String Filter selected users on the user type.
is_leader FilterGroupType Only returns users who are leaders of that type of group

Return value

This method returns a JSON structure. An array, all elements are of type SimpleUser

Name Type Description
SimpleUser Each element of the Array

Errors

HTTP Status Name Optional detail Description
400 bad_request get_parameter_error At least one of the GET parameters is wrong or missing.
